🎓 Degrees of Difference: What Impacts Earnings 10 Years After Graduation?

Data analytics capstone project using R, SQL, Tableau, and machine learning to analyze the institutional and financial factors influencing long-term graduate earnings across U.S. colleges.

📊 Project Overview

This project explores which institutional characteristics most strongly predict graduates’ earnings ten years after college entry.

The analysis examines:

Institution type
Geographic region
Program type (Business vs Healthcare)
Degree level
Graduate debt
Retention rates
Pell Grant percentages
Early-career earnings

Using the U.S. Department of Education College Scorecard dataset, we built statistical and machine learning models to identify the strongest drivers of long-term salary outcomes.

🔍 Key Findings

Early-career earnings were the strongest predictor of long-term salary.
Graduate debt showed a positive relationship with future earnings.
Higher Pell Grant percentages were associated with lower long-term earnings.
Degree level had stronger predictive power than program type.
Geographic region became less important after controlling for financial variables.
Final Log-OLS model achieved an Adjusted R² of approximately 0.877.
